# Moniker for Meteor

Meteor package for [Moniker](https://github.com/weaver/moniker) - A random name generator for Node.js.


## Installing

```bash
    meteor add xinranxiao:moniker
```

## Usage

```js
  console.log(Moniker.choose());
  // Example output: murky-hands
```
